# Getting Started with MCP Vector Search

Welcome! This guide will help you get up and running with MCP Vector Search in minutes.

## Contents

- [Installation Guide](./installation.md) — Install MCP Vector Search via pip, uv, or from source
- [Configuration Guide](./configuration.md) — Configure project settings, file extensions, and indexing behavior
- [First Steps: Quick Start Guide](./first-steps.md) — Index your first project, run your first search, and learn common workflows

## Quick Start

```bash
# 1. Install
pip install mcp-vector-search

# 2. Initialize your project
mcp-vector-search install

# 3. Search your code
mcp-vector-search search "authentication logic"
```
